Java是强类型语言,每个变量都需要声明其数据类型.
Java数据类型可分为基本数据类型和引用数据类型.

# 基本数据类型

## 数值型
### 整数类型
`byte` : 一个字节 
`short` : 两个字节
`int` : 四个字节
`long` : 八个字节
